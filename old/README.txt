how to compile:

1.	open .fig (figma) file and export it using this (https://www.figma.com/community/plugin/1417788739345122322/svg-export-with-text) plugin
2.	you now should have .svg file
4.	run "python export_from_svg.py" and input the .svg file name (you can edit the script to translate if you wish to)
5.	your files should be ready (YOUR_PERIODIC_TABLE.png, YOUR_PERIODIC_TABLE.svg)
